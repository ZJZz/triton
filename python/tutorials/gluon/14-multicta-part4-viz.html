第四部分 · TMA 多播（get_cga_layout：A/B 的广播布局）

A 沿 M 切 B 沿 N 切 multicast
由 C 的 cga_layout 推出 A、B：A 保留 M、清零 N；B 保留 N、清零 M。 被清零的维度上，不同 CTA 加载的是完全相同的数据 → 用 TMA multicast 让它们共享一次 L2 取数。 切换 cga_layout、把鼠标移到 中间分块图（A 行块 / B 列块 / C-tile）→ 看各 CTA 分工，右侧③联动 L2 收益。
① get_cga_layout：从 C 推 A、B
② matmul 全局分块：各 CTA 负责哪块 · 用哪部分 A/B 悬停 A 行块/B 列块/C-tile
③ 为什么省：L2 多播 随②悬停联动 · 有/无对照
④ 最小示例 tma_multicast_copy_kernel（cga=[[0,0]]，两 CTA 收同一 tile） 非 2CTA → barrier 普通
